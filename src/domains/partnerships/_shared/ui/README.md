# Shared UI Library

Central place for primitives that every partnership feature can consume.

## Marketing Patterns

- Located under `shared/ui/marketing-patterns/*`.
- Provide reusable animated backgrounds (e.g., `FallingPattern`) and demo wrappers.
- Keep them purely presentational—no feature-specific data or hooks.
- Import via `@/domains/partnerships/_shared/ui/marketing-patterns/FallingPattern`.
