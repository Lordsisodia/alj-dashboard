# Academy Shared

Cross-page components, data/services, and shared UI shells live here.

Subfolders:
- `application/`, `domain/`, `infrastructure/`, `data/`, `docs/` – non-UI layers.
- `components/` – shared React pieces (e.g., `LearningHubResponsive`).
- `views/desktop/`, `views/mobile/` – platform-specific shells and screens grouped under `views/` to keep the root tidy.
